centos7部署ServerStatus-Rust版探针

以针会友，分享下在吃灰小鸡上部署ServerStatus-Rust版探针的过程

服务端

1.在opt目录新建ServerStatus文件夹并进入

mkdir -p /opt/ServerStatus && cd /opt/ServerStatus

2.如果是服务端就下载服务端所需要的文件

wget --no-check-certificate -qO server.zip  "https://pan.11.yt/chfs/shared/ServerStatus-Rust/server.zip"

3.解压文件

unzip -o server.zip

4.给文件加755权限

chmod 755 stat_server stat_client

5.修改配置文件config.toml里面的端口和hosts参数，根据需求修改

vim config.toml

摁i进入插入模式进行编辑，编辑完摁esc退出编辑进入命令行模式，摁冒号：进入底行模式，输入wq保存并退出，q是退出，q!是不保存强制退出

6.移动stat_server.service和stat_client.service到系统服务文件夹

mv -v stat_server.service /etc/systemd/system/stat_server.service
mv -v stat_client.service /etc/systemd/system/stat_client.service

7.重新加载系统服务

systemctl daemon-reload

8.启动服务

systemctl start stat_server
systemctl start stat_client

9.查看状态

systemctl status stat_client

10.使用以下命令开机自启

systemctl enable stat_server
systemctl enable stat_client

11.停止服务

systemctl stop stat_server
systemctl stop stat_client

客户端

1.在opt目录新建ServerStatus文件夹并进入

mkdir -p /opt/ServerStatus && cd /opt/ServerStatus

2.如果是客户端就下载客户端所需要的文件

3.解压文件

4.给文件加755权限

chmod 755 stat_client

5.修改stat_client.service文件，将tcp://127.0.0.1:34512修改为http://你的服务端ip:端口/report

vim stat_client.service

摁i进入插入模式进行编辑，编辑完摁esc退出编辑进入命令行模式，摁冒号：进入底行模式，输入wq保存并退出，q是退出，q!是不保存强制退出

6.移动stat_client.service到系统服务文件夹

mv -v stat_client.service /etc/systemd/system/stat_client.service

7.重新加载系统服务

systemctl daemon-reload

8.启动服务

systemctl start stat_client

9.查看状态

systemctl status stat_client

9.使用以下命令开机自启

systemctl enable stat_client

10.停止服务

systemctl stop stat_client
